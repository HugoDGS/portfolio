# Portfolio

Personal portfolio built with Next.js 15, Tailwind CSS and Framer Motion.

## Stack

- **Framework** — Next.js 15 (App Router)
- **Styling** — Tailwind CSS 3
- **Animations** — Framer Motion 11
- **Language** — TypeScript
- **Deploy** — Vercel

## Sections

- Hero with animated gradient and scroll hint
- Projects — 6 featured projects with category filtering
- Skills — grouped by Frontend / Backend / DevOps / Security
- About — bio and stats
- Contact — GitHub, LinkedIn, email

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
